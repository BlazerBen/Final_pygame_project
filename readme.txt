   
Requirements
installments(pygame 2.3.0 , Python 3.10.11)
Run from director.py


Goals
enemies spawn when a previous enemy dies, don't let the enemies get close, 
kills count in top right corner, get the highest kill number, the game is over when health runs out

Controls
click on buttons to activate
wasd: up left down right(normal movement)
shift: sprint(move at 1.5 speed)
click: shoot and use mana(wait to refill mana)
escape: Pause
